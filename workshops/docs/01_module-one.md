# Climate Analysis

The foundation of sustainable and environmental-driven design starts with a comprehensive understanding of climates. To ensure a building responds to its environment, we should start with analyzing high-fidelity EPW (EnergyPlus Weather) weather data.

The weather data provides important information about the temperature, humidity, cloud coverage, etc... of a location, solar radiation, which are helpful for knowing the microclimates of your site.

In this module, we will learn where to find weather files and how to import them into ClimateStudio to visualize key aspects of climate that inform design choices.

## Where can I find weather files?
Weather data is most commonly stored in .epw files. You can get .epw files from three main websites:

1. <a href="https://climate.onebuilding.org/" target="_blank">climate.onebuilding.org</a>: more recent weather files.
2. <a href="https://energyplus.net/weather" target="_blank">Energyplus website</a> : hosted by the energyplus software website.
3. <a href="https://www.ladybug.tools/epwmap/" target="_blank">ladybug epw map</a> : created by Ladybug Tools.

In this tutorial, we will use the weather files from climate.onebuilding.org as they are more up to date.
